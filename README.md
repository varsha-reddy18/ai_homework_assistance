# 🎓 AI-Powered Homework Assistance System

An intelligent **AI-based educational web application** designed to help students with homework, grammar correction, study planning, quizzes, flashcards, voice interaction, and general knowledge learning.

This project is developed as a **B.Tech Mini Project** for exploring the use of **Artificial Intelligence in Education**.

---

## 📌 Project Overview

The **AI-Powered Homework Assistance System** is a smart learning platform that helps students understand concepts, improve grammar, plan studies, revise using flashcards, and interact with AI in an engaging way.

The system combines:

- 🤖 AI-powered question answering  
- ✍️ Grammar learning and correction  
- 📅 Smart study planner  
- ❓ Quiz and flashcards  
- 🌍 General knowledge explorer  
- 🎤 Voice input and speech output  
- 📷 Image-based homework support using OCR  

This platform is built to make learning **interactive, accessible, and student-friendly**.

---

## 🚀 Features

### 🏠 Dashboard
A clean and modern student dashboard with navigation to all learning tools.

### 🤖 Ask AI Assistant
Students can ask questions subject-wise and receive AI-generated answers.

Supported subjects include:
- Mathematics
- Science
- Social
- English
- Telugu
- Hindi
- General Knowledge

### ✍️ Grammar Checker
Students can improve English grammar and learn important grammar topics such as:

- Tenses
- Parts of Speech
- Active and Passive Voice
- Punctuation
- Sentence Structures
- Vocabulary
- Articles
- Subject-Verb Agreement

### 📅 Smart Study Planner
Students can enter previous marks and receive:

- Performance classification:
  - 🌟 Excellent
  - ✅ Good
  - ⚠️ Average
  - ❌ Poor
- Personalized improvement tips
- Suggested daily and weekly study plans

### ❓ Quiz Module
Interactive quiz section for self-assessment and practice.

### 🧠 Flashcards
Quick revision tool for improving memory and concept recall.

### 🎮 Puzzle / Learning Activities
Fun educational engagement tools for students.

### 🌍 General Knowledge Section
Includes knowledge cards and learning modules for:

- Countries and Capitals
- Indian States and Capitals
- Continents
- Oceans
- Rivers
- Deserts
- Science Facts

### 🎤 Voice Support
- Speech-to-text (voice input)
- Text-to-speech (voice output)

### 📷 Ask From Image
Students can upload an image containing homework text and ask questions based on it.

## ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/varsha-reddy18/ai_homework_assistance.git
cd HOMEWORK-ASSISTANCE
```



---

## 2️⃣ Backend Setup

Move to backend folder:

```bash
cd backend
```

Create a virtual environment:

```bash
python -m venv venv
```

### Activate virtual environment

#### Windows

```bash
venv\Scripts\activate
```


Install dependencies:

```bash
pip install -r requirements.txt
```

Run FastAPI server:

```bash
python -m uvicorn main:app --reload
```

Backend will run at:

```bash
http://127.0.0.1:8000
```

Swagger API docs:

```bash
http://127.0.0.1:8000/docs
```

---

## 3️⃣ Frontend Setup

Go to frontend folder and open the project using:

* **VS Code Live Server**
  OR

## 🌐 API Endpoints

### 🔐 Authentication

* `POST /signup` → User Signup
* `POST /login` → User Login

### 🤖 AI Features

* `POST /ask` → Ask subject-based questions
* `POST /ask-from-image` → Ask questions from uploaded image
* `POST /grammar-check` → Grammar correction

---

## 🧪 Example API Request

### `/ask`

```json
{
  "question": "What is photosynthesis?",
  "subject": "Biology",
  "session_id": "default",
  "user_id": "student123"
}
```

### Example Response

```json
{
  "answer": "Photosynthesis is the process by which green plants prepare their food using sunlight, carbon dioxide, and water."
}
```
